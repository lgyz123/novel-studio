# Continuity Checklist

Use this checklist to derive the smallest possible set of writer guardrails.

## State

- Current book time
- Relative order vs previous scene
- Protagonist mode
- Investigation stage
- Risk level

## Artifacts

- Holder
- Location
- Visibility
- Whether the item changed in recent scenes

## Relationships

- Current stance between key characters
- Whether a suspicion has become confirmation
- Whether pressure is ambient or direct

## Scene bridge

- What the last locked scene changed
- What cannot be silently reset
- Which new consequence must still be visible

## Repair priority

For revise/rewrite rounds, check in this order:

1. Artifact drift
2. Timeline drift
3. Investigation-stage drift
4. Relationship drift
5. Risk-level drift
