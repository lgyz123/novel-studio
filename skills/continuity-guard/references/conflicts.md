# Common Continuity Conflicts

## Artifact reset

Pattern:

- tracker says an item is carried on body
- draft writes it as hidden in a box, shelf, or room again

Response:

- preserve tracker state
- rewrite the scene action around the current artifact position

## Time blur

Pattern:

- chapter state says `次日` or `白天`
- draft still reads like the same night

Response:

- make the time marker explicit early
- add one physical sign of time change

## Risk rollback

Pattern:

- prior scene escalated risk from `high` to `critical`
- draft writes the protagonist as if pressure returned to normal

Response:

- keep at least one external sign of elevated risk
- do not reduce risk silently

## Investigation rollback

Pattern:

- prior scene changed investigation from passive memory to active probing
- draft writes the protagonist as if nothing shifted

Response:

- preserve the newer action posture
- even if the scene quiets down, keep the changed behavior visible
