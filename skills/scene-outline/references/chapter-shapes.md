# Chapter Shape Patterns

Use these as planning references, not rigid formulas.

## Tight pressure chapter

1. establish unstable normal
2. introduce disturbance
3. test or probe
4. encounter resistance
5. end on changed handling

## Slow-burn suspicion chapter

1. present ordinary action
2. let old detail resurface in a new context
3. move from internal discomfort to external probing
4. produce a small but irreversible consequence

## Repair usage

If reviewer says the chapter is stalling:

- prefer a consequence or decision scene next
- avoid adding another atmosphere-only bridge
