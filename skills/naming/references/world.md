# World Naming

Use these questions:

- Is this name for a place, organization, artifact, or technique?
- Should it sound official, vernacular, sacred, feared, or commercial?
- Does it come from local speech, institutional naming, or inherited older language?

## Good outputs

- 5 to 10 compact candidates
- each with a style tag and best use

## Avoid

- names that imply a whole new cosmology unless the manifest already supports it
- empty grandeur with no tonal grounding
